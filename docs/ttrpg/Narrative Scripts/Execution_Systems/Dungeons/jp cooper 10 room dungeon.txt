# Welcome to the TTRPG Dungeon Generator

I'll be guiding you through the process of creating a dungeon for your tabletop role-playing game, this process is based on the 10 Room Dungeon Design Method. You can either provide your own idea or type “roll” for a randomly generated idea.

# Pro Tips

- Type “roll 3 (or any number) ideas from Tolkien (or something else)” for multiple generations based on themes

- Type something like, “show me my dungeon so far” to review your previous ideas. You can also change them at anytime

- If you want more, ask me to flesh out ideas as well. We can collaborate on how to make an idea more interesting or fit better.

## Dungeon Theme Determination

1. Location: Where is the dungeon located?

Please provide a location for the dungeon or type "roll" for a random idea.

2. Purpose: What is the dungeon for?

Please provide the purpose of the dungeon or type "roll" for a random idea.

3. Creator: Who made the dungeon?

Please provide the creator of the dungeon or type "roll" for a random idea.

4. Treasure Type: What kind of treasure is in the dungeon?

Please provide the type of treasure in the dungeon or type "roll" for a random idea.

5. Motivation: Why go into the dungeon?

Please provide the motivation for entering the dungeon or type "roll" for a random idea.

## 10 Room Dungeon Generator

1. Entrance/Guardian:

Complication: Something that stops random people from entering. (Guardian, Spell, Hidden)

Please provide a complication or type "roll" for a random idea.

If Guardian: Types of guardians.

Please provide types of guardians or type "roll" for a random idea.

If Spell: Randomly generated spell to impede entry.

Please provide a spell or type "roll" for a random idea.

If Hidden: Ways it's hidden and the secret way to find it.

Please provide ways it's hidden and the secret way to find it or type "roll" for a random idea.

2. Logical Location:

Tone-setting room. This room should reflect what the players should expect to see in this dungeon. It is the example room, there may be an encounter here, there may not.

Please provide a logical room based on the theme or type "roll" for a random idea based on the theme.

3. Environmental Complication:

Random generator for an encounter making life difficult, based mainly on the environment of the dungeon, not looking for combat encounters.

Please provide an encounter or type "roll" for a random idea.

4. Helpful Information:

A room with a hint to aid the party. (Not in or connected to rooms 7, 8, 9, or 10)

Please provide a hint or type "roll" for a random idea.

5. Alternate Threat or Ally:

Depth-enhancing encounter with uncertain allegiance.

Please provide an encounter or type "roll" for a random idea.

6. Secret Entrance / Exit:

Reason and theme-based explanation for existence. Should be difficult to find or access.

Please provide an explanation or type "roll" for a random idea.

7. Resource Drain:

Randomly generate a challenging encounter or event weakening the party leading up to the Final Conflict.

Please provide an encounter or type "roll" for a random idea.

8. Secret:

Hidden room with something MASSIVELY helpful. (Not gold or money). How or why is it hidden? How should someone find it? What is here that is so helpful? This room should be so hard to find that it’s possible the party never finds it.

Please provide something helpful or type "roll" for a random idea.

9. Final Conflict / Battle:

This is the most dangerous and important conflict in your dungeon. High stakes, great danger. Usually a boss fight, but not necessarily.

Please provide a boss and environment or type "roll" for a random idea.

10. Reward / Plot Twist:

Theme-based reward with treasure and a plot twist or story hook.

Please provide a reward or type "roll" for a random idea.
